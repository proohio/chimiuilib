Creating Window

Create the main window for your UI library. This is the foundation of your interface.

Syntax

Lua 
local Window = Library.CreateLib(title, theme)

Parameters

title
string
The window title displayed in the header
theme
string | table
"Chimi", "DarkTheme", "LightTheme", "Ocean", or custom table

Example

Lua 
-- Create window with Chimi theme
local Window = Library.CreateLib("My Script", "Chimi")

-- Create window with custom theme
local customTheme = {
    SchemeColor = Color3.fromRGB(255, 255, 255),
    Background = Color3.fromRGB(12, 12, 12)
}
local Window = Library.CreateLib("Custom UI", customTheme)

Tabs & Sections

Organize your UI into tabs and sections for better structure.

Creating Tabs

Lua 
local Tab = Window:NewTab(tabName)

Creating Sections

Lua 
-- Regular section
local Section = Tab:NewSection("Section Name")

-- Hidden header section
local Section = Tab:NewSection("Hidden", true)

Button

Create clickable buttons that execute functions.

Lua 
local button = Section:NewButton("Button Name", "Description", function()
    print("Button clicked!")
end)

-- Update button text
button:UpdateButton("New Text")

Toggle

Create toggle switches with on/off states.

Lua 
local toggle = Section:NewToggle("Feature", "Enable feature", function(state)
    if state then
        print("Enabled!")
    else
        print("Disabled!")
    end
end)

-- Update toggle
toggle:UpdateToggle("New Text", true) -- Set to enabled

Slider

Create sliders for numeric input with min/max values.

Lua 
Section:NewSlider("Speed", "Walk speed", 200, 16, function(value)
    game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = value
    print("Speed set to:", value)
end)

TextBox

Create text input fields for user input.

Lua 
Section:NewTextBox("Username", "Enter your name", function(text)
    print("You entered:", text)
end)

Dropdown

Create dropdown menus with multiple options.

Lua 
local options = {"Option 1", "Option 2", "Option 3"}
local dropdown = Section:NewDropdown("Select", "Choose option", options, function(selected)
    print("Selected:", selected)
end)

-- Refresh dropdown options
local newOptions = {"New 1", "New 2", "New 3"}
dropdown:Refresh(newOptions)

Label

Create text labels for displaying information.

Lua 
local label = Section:NewLabel("Status: Active")

-- Update label text
label:UpdateLabel("Status: Updated")

Custom Themes

Create your own custom color schemes.

Lua 
local customTheme = {
    SchemeColor = Color3.fromRGB(255, 255, 255),
    Background = Color3.fromRGB(12, 12, 12),
    Header = Color3.fromRGB(8, 8, 8),
    TextColor = Color3.fromRGB(255, 255, 255),
    ElementColor = Color3.fromRGB(18, 18, 18),
    AccentColor = Color3.fromRGB(240, 240, 240)
}

local Window = Library.CreateLib("Custom UI", customTheme)
